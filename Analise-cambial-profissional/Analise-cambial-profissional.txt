ETAPAS DE EXECUÇÃO:

   - Selecionar mesmo intervalo de datas: 18/07/2025 a 18/01/2026.

3. CONSOLIDAÇÃO EM PLANILHA ÚNICA
   - Abrir Excel/Google Sheets.
   - Criar aba "Dados Consolidados".
   - Colunas sugeridas:

     | Data | BRL/USD (Compra) | BRL/USD (Venda) | Spread | BRL/USD Média |
     | PYG/USD (Compra) | PYG/USD (Venda) | PYG/USD Média |
     | PYG/BRL (Compra) | PYG/BRL (Venda) | PYG/BRL Média |
     | Variação diária (%) | Retorno log (%) |
     | Média móvel 7d | Média móvel 30d |
     | Volatilidade 30d | Máx/Mín acumulado |
     | Sharpe diário (com Rf real) | Sharpe anualizado |
     | Drawdown Máx | Beta Cambial |
     | VaR (95% empírico) | VaR (95% paramétrico CF) |
     | Taxa de juros BR | Taxa de juros PY |
     | Inflação BR | Inflação PY |
     | Volume de negociação (se disponível) |
     | Eventos macro relevantes (COPOM, BCP, etc.) |

   - Fórmulas principais:
     * Spread = Venda - Compra
     * Média = (Compra + Venda) ÷ 2
     * PYG/BRL = PYG/USD ÷ BRL/USD
     * Variação diária (%) = (Cotação hoje - Cotação ontem) ÷ Cotação ontem × 100
     * Retorno log = ln(P_t / P_t-1)
     * Média móvel = MÉDIA(intervalo)
     * Volatilidade = DESVPAD(intervalo)
     * Sharpe diário = (Retorno diário - Rf diário) ÷ Volatilidade diária
     * Sharpe anual = Sharpe diário × √252
     * Drawdown = Máx acumulado - Valor atual
     * Beta = COV(PYG, BRL) ÷ VAR(BRL)
     * VaR empírico = Percentil 5% da série de retornos
     * VaR CF = μ + σ * z_CF (Cornish-Fisher)
       - z_CF ≈ z + (S/6)(z² - 1) + (K/24)(z³ - 3z) - (S²/36)(2z³ - 5z)
     * Taxa de juros = conforme dados oficiais (SELIC, BCP)
     * Inflação = IPCA (Brasil), IPC-PY (Paraguai)
